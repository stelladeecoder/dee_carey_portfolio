Credit card fraud detection (GBM)

Machine learning · Finance risk

Detecting fraudulent credit card transactions using a Gradient Boosting model on an imbalanced dataset.

  • Built a full pipeline for the Kaggle fraud dataset: cleaned features, scaled inputs, and used a stratified split to preserve class balance.
  • Trained a GradientBoostingClassifier and evaluated precision, recall, F1 score, and ROC AUC.
  • Benchmarked against a Random Forest baseline and achieved stronger performance on the fraud class.
  • Documented lessons learned and ideas for next steps like SMOTE and anomaly detection.

Stack: Python, Pandas, NumPy, scikit-learn, Matplotlib, Seaborn, Kaggle
