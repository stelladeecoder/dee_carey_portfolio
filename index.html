Credit Card Fraud Detection GBM

Machine learning · Finance risk

Detecting fraudulent credit card transactions using a Gradient Boosting model on an imbalanced dataset.

  • Built a full data pipeline for the Kaggle Credit Card Fraud Detection dataset: cleaned features, scaled inputs, and used a stratified split to keep class balance.
  • Trained a GradientBoostingClassifier and evaluated precision, recall, F1 score, and ROC AUC.
  • Benchmarked against a Random Forest approach and achieved very strong performance, with near perfect ROC AUC on the fraud class.
  • Summarized lessons learned and identified next steps like SMOTE and anomaly detection.

Stack: Python, Pandas, NumPy, scikit learn, Matplotlib, Seaborn, Kaggle
